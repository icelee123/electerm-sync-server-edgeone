# Electerm 同步服务器 (EdgeOne KV)

[English](README.md) | [中文](README_CN.md)

## 环境变量

| 变量 | 必填 | 描述 |
|------|------|------|
| `JWT_SECRET` | 是 | JWT 签名的密钥 |
| `JWT_USERS` | 是 | 允许用户的逗号分隔列表 |
| `KV_BIND_VAR` | 是 | EdgeOne KV 绑定变量名 |
